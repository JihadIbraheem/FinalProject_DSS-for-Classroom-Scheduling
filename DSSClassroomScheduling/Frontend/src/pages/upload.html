File Requirements:
• Must be Excel (.xlsx)
• Required columns: יום, חדר, בניין, קיבולת
• These columns must not have empty cells: יום, חדר, בניין
• Time slot columns that must exist: 08:00 - 09:00, 09:00 - 10:00, 10:00 - 11:00,
11:00 - 12:00, 12:00 - 13:00, 13:00 - 14:00,
14:00 - 15:00, 15:00 - 16:00, 16:00 - 17:00,
17:00 - 18:00, 18:00 - 19:00, 19:00 - 20:00,
20:00 - 21:00, 21:00 - 22:00
• If a time cell contains data, it must be in this format:
א-214.1300א01( 144)[ד"ר ברקן ניצנה-ויקטוריה]{הסתברות למערכות מידע - שיעור}
• Course code 214.1300א01, number of students (144), and course name {...} are required.
• Instructor name [...] is optional.
• The classrooms in the file must match the existing classrooms in the database (you can check or edit the available classrooms on the "Manage Classrooms").
